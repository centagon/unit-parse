# Unit Parse 

[![Build Status](https://travis-ci.org/centagon/unit-parse.svg?branch=master)](https://travis-ci.org/centagon/unit-parse)

```js
const parseUnit = require('unit-parse');

console.log(parseUnit('20px'));
// { value: '20', unit: 'px' }
```
